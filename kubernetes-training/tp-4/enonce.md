#### TP4: GESTION DU STOCKAGE (minikube ) 

VOLUME
```bash
kubectl apply -f mysql-volume.yml
```
```bash
kubectl get po -o wide
```
```bash
kubectl port-forward mysql-volume 3306:3306 --address 0.0.0.0
```

APPLY VOLUME
```bash
kubectl get po -o wide
```
```bash
kubectl delete  -f mysql-volume.yml
```
```bash
kubectl get po -o wide
```

